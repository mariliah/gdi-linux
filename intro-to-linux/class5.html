Networking

Networking refers to the practice of connecting computers and other devices to share resources and information. This connection can be through wired or wireless means and enables communication between devices locally (within a small geographic area, like a home or office) or over larger distances (across cities, countries, or even globally via the internet).

Key concepts

  • IP addressing (IPv4, IPv6)
  • Subnetting
  • Gateways
  • Domain Name System (DNS)
  • Network Interface Card (NIC)
  • MAC Address

IP Addressing

  • IPv4: Uses 32-bit addresses (e.g. 192.168.1.1) and is the most widely used version.
  • IPv6: Uses 128-bit addresses (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334) to provide a larger address space and improve security and performance.

Subnetting

Divides a larger network into a smaller, manageable segments (subnets), improving efficiency and security.

Gateways

Devices that route traffic from a local network to other networks or the itnernet. The default gateway typically connects a local network to the broader internet.

DNS

Translates human-readable domain names (e.g., www.google.com) into IP addresses that computers use to identify each other on the network.

NIC

Hardware that connects a computer to a network, either via cable (Ethernet) or wirelessly (Wi-Fi).

NIC

MAC Address

A unique identifier assigned to a NIC. It is used for communication within a local network segment.

Networking in practice

  • Devices (computers, printers, smartphones) are connected to a network through NICs.
  • Each device is assigned an IP address, either manually (static) or automatically via DHCP (Dynamic Host Configuration Protocol).
  • Devices communicate with each other using IP addresses, and data is routed through gateways to reach other networks or the internet.
  • DNS servers are used to resolve domain names to IP addresses, allowing users to access websites using human-readable names.
