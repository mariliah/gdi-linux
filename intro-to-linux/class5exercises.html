Class 1 Exercises

Exercise #1

  1. Run the `ifconfig` command to display network interfaces
  2. Identify the IP address, netmask, and MAC address of the primary network interface

Exercise #2

  1. Run the `ip a` command to show detailed information about network interfaces
  2. Note the same details: IP address, netmask, and MAC address

Exercise #3

  1. Run the `ping` command to check connecticity to a known website
  2. ping google.com
  3. ping the local router: 192.168.1.1

Exercise #4

  1. Use the `ifconfig` command to assign a static IP address to the primary interface (replace `eth0` with the correct interface name if different)
  2. sudo ifconfig eth0 192.168.1.100 netmask 255.255.255.0
  3. Verify the new IP address by running `ifconfig` again
