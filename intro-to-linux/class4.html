What is Bash?

Bash (Bourne Again SHell) is a command-line interpreter or shell for Unix-based operating systems. It is an enhancement of the original Bourne Shell (sh), developed by Stephen Bourne. Bash is the default shell for many Linux distributions

It used to be the default shell for macOS systems before Apple decided to change to the Z shell.

What is Bash? (continued)

  • Bash allows users to execute commands to interact with the operating systems.
  • Commands can be simple, like `ls` to list directory contents, or complex, involving pipelines and command chaining.
  • Bash provides job control features, allowing users to run commands in the background, bring them to the foreground, and manage multiple tasks simultaneously.

What is Bash scripting?

  • Bash scripting is the act of creating text files containing a series of commands that are executed sequentially.
  • Scripts automate repetitive tasks, manage system configurations, and handle batch processing.
  • Programming constructs: Bash supports control flow constructs such as if statements, loops (for, while), and functions, making it a powerful scripting language for automation.

When/Why to use Bash scripting?

  • To automate simple tasks.
  • Lightweight and always available.
  • Although there are more advanced tools/programming languages(Ansible and Python respectively), having a basic understanding of Bash scripting makes a big difference.

Simple Bash Script

                
#!/bin/bash

# Check if the script is run as root
if [[ $(id -u) ne 0]]; then
echo "This script must be run as root."
exit 1
fi

# Function to add user to sudoers file
add_users_to_sudoers() {
    username="$1"
    if grep -q "^username" /etc/passwd; then
        # Check if the user is already in sudoers file
        if grep -q "^$username" /etc/sudoers; then
        echo "$username is already in sudoers file."
        else
        # Add the user to sudoers File
        echo "$username ALL=(ALL) ALL >> /etc/sudoers"
        echo "User $username added to sudoers file."
    fi
    else
        echo "User $username does not exist."
    fi
}

# Main script starts here
if [[ $# -eq 0]]; then
echo "Usage: $0 "
    exit 1
fi

    username="$1"
    add_users_to_sudoers "$username"

Script Parameters

In Bash scripting, parameters (also known as arguments) are used to pass information to a script or a function. These parameters allow scripts to be dynamic and flexible.

                    
                    #!/bin/bash
                    # Simple calculator
                    
                    # Store the first parameter in num1 variable
                    num1=$1

                    # Store the second parameter in num2 variable
                    num2=$2

                    # Store the addition results in the total variable
                    TOTAL=$(($num1 + $num2))

                    echo '###########'
                    printf "%s %d\n" "The total is=" $TOTAL
                    echo '###########'

User Input

Another way to interact with the supplied input from the shell script is to use the `read` function. 
                
                    #!/bin/bash
                    read -p "Please enter your first name:" FIRSTNAME
                    read -p "Please enter your last name:" LASTNAME

                    printf "Your full name is: $FIRSTNAME $LASTNAME\n"
